About This Flask App

This is a sample Flask application demonstrating basic web development concepts.

Features

  • ✅ Responsive Bootstrap UI
  • ✅ Template inheritance with Jinja2
  • ✅ RESTful API endpoints
  • ✅ Google Sheets keyword search tool
  • ✅ Web scraping with BeautifulSoup
  • ✅ Multi-threaded URL processing
  • ✅ Error handling
  • ✅ Static file serving
  • ✅ JSON data handling

Technology Stack

Backend
  • Python 3.x
  • Flask 2.3.3
  • Jinja2 Templates
  • Werkzeug WSGI
  • Google Sheets API
  • BeautifulSoup4
  • Requests library
Frontend
  • HTML5
  • Bootstrap 5
  • JavaScript (ES6+)
  • CSS3
Getting Started

To run this application:

  1. Install dependencies: pip install -r requirements.txt
  2. Run the app: python app.py
  3. Open your browser to: http://localhost:5000
